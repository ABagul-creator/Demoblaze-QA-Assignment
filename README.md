# Demoblaze QA Assignment

## Project Overview
This repository contains manual testing artifacts created for the Demoblaze e-commerce application.

## Scope of Testing
- User Signup
- Login
- Product Browsing
- Cart Management
- Checkout Process
- Logout
- Contact & About modules
- Non-functional testing (Compatibility, Accessibility)

## Deliverables
- Test Case Document
- Test Execution Report
- Defect Report with Severity & Priority
- Test Data Sheet
- Explanation Video (MP4)

## Tools Used
- Manual Testing
- MS Excel
- GitHub for documentation

## Video Explanation
A detailed screen recording explaining test cases, execution, defects, and approach is included in the Video folder.
